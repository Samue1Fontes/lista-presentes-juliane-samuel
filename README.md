Uma landing page / convite digital criada para o chá de casa nova de Juliane e Samuel.

Este projeto foi desenvolvido como um site real para ser utilizado no evento e também como um projeto de estudo em front-end, explorando tecnologias modernas, animações e boas práticas de performance.

O objetivo foi criar uma página leve, visual e responsiva, reunindo informações do evento (data, horário e local) e uma lista de presentes, além de pequenos detalhes visuais como polaroids e animações para tornar a experiência mais agradável.

---

## Visão Geral

A aplicação apresenta:

- Informações do evento (data, horário e local);
- Lista de presentes organizada e intuitiva;
- Elementos visuais decorativos (polaroids, animações e transições suaves);
- Layout responsivo para mobile, tablet e desktop;
- Mapa integrado com o local do evento.

## Sobre o Desenvolvimento

O projeto foi construído com foco em organização de componentes, animações bem controladas e boa performance.

Durante o desenvolvimento, foi possível trabalhar e aprender na prática com tecnologias que ainda não faziam parte do uso cotidiano, como React, Vite e GSAP, além de aprofundar conceitos relacionados a animações, responsividade e otimização de assets.

A estrutura foi pensada para ser simples, clara e fácil de manter, com atenção especial à experiência do usuário.

---

## Rodando o Projeto Localmente

### Pré-requisitos

- Node.js
- npm

### Passos

1. Clone o repositório:

git clone <URL-do-repo>
cd lista-presentes-juliane-samuel

2. Instale as dependências:

npm install

3. Execute o projeto em ambiente de desenvolvimento:

npm run dev

4. Build para produção:

npm run build

5. Preview do build:

npm run preview
